# azlib-language
WIP Language lib for me :D


To add in your maven project

**Repository:**
```xml
<repository>
  <id>jitpack.io</id>
  <url>https://jitpack.io</url>
</repository>
```

**Dependency:**
```xml
<dependency>
  <groupId>com.github.Azrotho</groupId>
  <artifactId>azlib-language</artifactId>
  <version>Tag</version>
</dependency>
```
